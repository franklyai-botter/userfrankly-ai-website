# Neural Nautic — Website UI Kit

A high-fidelity interpretation of the Neural Nautic marketing site. Components are modular React/JSX, intended to be mixed and matched.

## Screens demonstrated in `index.html`
1. **Home** — hero over neural-net bg, featured collection, atelier note
2. **Collection** — editorial grid of goods
3. **Journal** — the atelier's log entries
4. **Detail** — a single object's chart

## Components
- `Header.jsx` — top navigation with brand lockup
- `Footer.jsx` — deep-sea footer with watermark mesh
- `Hero.jsx` — full-bleed neural-network hero
- `FeaturedCollection.jsx` — editorial product grid
- `JournalList.jsx` — log entries, editorial rhythm
- `AtelierNote.jsx` — the "about" voice moment
- `ObjectCard.jsx`, `LogEntry.jsx` — primitive cards
- `Button.jsx`, `Tag.jsx`, `Eyebrow.jsx` — atoms

> **Note:** No product/codebase was supplied; this is an interpretation derived from the brand's three source images. Please iterate.
