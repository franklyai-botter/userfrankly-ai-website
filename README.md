# NeuralNautic Website Handoff Package

Dieses Paket ist ein strukturiertes Übergabepaket für die Erstellung der NeuralNautic-Website mit Claude Code, Cursor oder einem ähnlichen Coding-Agenten.

## Ziel

Auf Basis des separat in Claude Design erstellten Designs soll eine professionelle B2B-Website für Frank Schütt / NeuralNautic umgesetzt werden.

## Positionierung

**Pragmatische KI-Beratung für den Mittelstand.**  
Fokus auf KI-Potenzialanalyse, Controlled Pilot, operative KI-Automatisierung, KI-Governance, Schulung und Tool-Auswahl.

## Zielgruppe

- Geschäftsführung im Mittelstand
- IT-Verantwortliche
- Fachabteilungen
- Datenschutzverantwortliche
- Betriebsräte
- Operative Teams
- Dienstleistungs-, Medien-, Event-, Backoffice- und technische Unternehmen

## Empfohlener Tech Stack

Falls noch nicht festgelegt:

- Next.js
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
- Framer Motion
- Lucide Icons
- Vercel Deployment

## Wichtige Dateien

- `PROJECT_BRIEF.md` – Gesamtbriefing für Entwickler und AI-Coding-Agenten
- `CONTENT_STRUCTURE.md` – Seiteninhalte je Navigation
- `content.json` – Maschinenlesbare Content-Struktur
- `CLAUDE_CODE_PROMPT.md` – Direkt nutzbarer Prompt für Claude Code
- `DESIGN_HANDOFF_NOTES.md` – Hinweise zur Übernahme des Claude-Designs
- `COMPONENT_PLAN.md` – Vorschlag für Komponenten, Sections und Routing
- `SEO.md` – SEO-Struktur, Meta-Texte und Keywords
- `GITHUB_ISSUES.md` – Umsetzbare GitHub-Issues / Arbeitspakete

## Umsetzungslogik

Erst die Startseite bauen, dann die Unterseiten.  
Design aus Claude Design übernehmen, aber Inhalte aus diesem Paket verwenden.

## Markenstil

Modern, ruhig, seriös, minimalistisch, mittelstandstauglich.  
Nicht zu verspielt. Kein generisches Roboter-Klischee.  
Kerngefühl: **kontrollierte Umsetzung statt KI-Hype.**
