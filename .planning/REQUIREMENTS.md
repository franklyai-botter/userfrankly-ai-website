# Requirements: NeuralNautic Website

**Defined:** 2026-05-01
**Core Value:** Ein potenzieller Kunde landet auf der Seite, versteht sofort was NeuralNautic macht und nimmt Kontakt auf.

## v1 Requirements

### Hero

- [ ] **HERO-01**: Neural Network im Hero der Startseite ist prominent sichtbar (Gradient zeigt das Netzwerk klar auf der rechten Seite)

### Animationen

- [ ] **ANIM-01**: Sektionen und Karten faden beim Scrollen in den Viewport ein (Scroll-Reveal via Intersection Observer)
- [ ] **ANIM-02**: Interaktive Elemente (Karten, Buttons, Links) haben subtile Hover-Effekte

### Mobile / Responsive

- [ ] **MOB-01**: Navigation ist auf Smartphone voll bedienbar (Hamburger-Menü oder Drawer)
- [ ] **MOB-02**: Hero-Section auf Smartphone: Schriftgrösse, Layout und Neural Network wirken gut
- [ ] **MOB-03**: Alle Grid-Layouts (3-spaltig, 4-spaltig, 5-spaltig) brechen auf Mobile korrekt in 1-2 Spalten zusammen
- [ ] **MOB-04**: Footer ist auf Mobile lesbar, Links sind tippbar (ausreichende Touch-Targets)
- [ ] **MOB-05**: Alle Unterseiten-Header (Neural Network Background) sehen auf Smartphone gut aus

## v2 Requirements

### Performance

- **PERF-01**: Lighthouse Performance Score > 90 auf Mobile
- **PERF-02**: Core Web Vitals im grünen Bereich

### SEO

- **SEO-01**: Strukturierte Daten (Schema.org) für lokales Unternehmen
- **SEO-02**: Open Graph / Twitter Card Meta-Tags für alle Seiten
- **SEO-03**: Sitemap.xml generiert

### Inhalt

- **CONT-01**: Echte Referenzen / Testimonials eingebaut
- **CONT-02**: Über-mich-Seite mit Foto

## Out of Scope

| Feature | Reason |
|---------|--------|
| Blog / CMS | Statischer Content reicht für v1 |
| Dark/Light Mode Toggle | Nur Dark Mode vorgesehen |
| i18n / Mehrsprachigkeit | Nur Deutsch |
| Animations Library (Framer Motion) | Erst mit nativem Intersection Observer lösen |
